# TaskTracker

A minimal CLI task tracking application for testing PR Context functionality.

## Installation

```bash
pip install -e .
```

## Usage

```bash
# Add a task
tasktracker add "Buy groceries"

# List tasks
tasktracker list

# Complete a task
tasktracker complete 1

# Delete a task
tasktracker delete 1
```

## Development

```bash
# Install dev dependencies
pip install -e ".[dev]"

# Run tests
pytest
```
